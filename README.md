# fsa-sig200

Basic shell script that shows how to poll SIG200 using the REST API, with an IF6123 sensor

## Dependencies
* JQ https://stedolan.github.io/jq/
* cURL

## Run
* Edit `config.env` 
* `./poll.sh`
